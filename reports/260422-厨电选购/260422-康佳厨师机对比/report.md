## Key Differentiators Summary

### Main Differences:

1. **Motor Technology:**
   - KM-917A & KM-918A: DC motors (quieter, more efficient, better for continuous operation)
   - KM-912A: AC motor (more powerful but noisier)

2. **Smart Features:**
   - KM-917A: Timer + Fermentation (unique advantage)
   - KM-918A & KM-912A: Basic mechanical controls only

3. **Capacity vs Power Trade-off:**
   - KM-918A: Small capacity (3L) with efficient DC motor
   - KM-912A: Large capacity (5L) with powerful AC motor
   - KM-917A: Best balance (5L + DC motor + smart features)

4. **Price Positioning:**
   - KM-918A: Entry-level (~¥459-500)
   - KM-917A: Mid-range with features (~¥559)
   - KM-912A: Higher price for power (~¥799)

The KM-917A represents the strongest competitive position in the sub-1000 RMB segment, offering DC motor technology and smart features (timer + fermentation) typically found in more expensive models.
