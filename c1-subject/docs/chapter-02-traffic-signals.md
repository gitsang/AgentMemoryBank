## 2.1 出题方式分析

本章题目有五种典型的出题套路：

**套路一：识图题——"这个标志是什么含义"**

> "这个标志是何含义？"（附图）

直接考标志、标线、手势图形的识别，是最常见的题型。不需要推理，记住样子就行。

**套路二：情景判断题——"图中行为是否正确"**

> "如图所示，在这种情况下只要后方没有来车，可以倒车。"（附图中单行路标志）

给你一幅实景图，判断驾驶员的行为是否符合标志标线规定。这种题很容易被场景迷惑——看起来"安全"不一定合法。

**套路三：信号灯通行规则——直问**

> "在路口遇有交通信号灯和交通警察指挥不一致时，按照交通信号灯通行。"

考你信号灯的基础通行规则，以及信号灯与交警指挥之间的优先级关系。

**套路四：对比混淆题——选项互相高度相似**

> "下列哪个交通标志表示不能停车？"（选项是禁止停放、禁止长时间停放、停车检查、停车让行的图示）

四个选项全是停车相关标志，全靠你能否区分"禁止停放"和"禁止长时间停放"的红叉/红斜杠差别。

**套路五：手势识别题——"这一组交警手势是什么信号"**

> "这一组交通警察手势是什么信号？"（附图）

8 种手势需要逐一识别，对"左/右"的区分是高频扣分点——左转手势和右转手势刚好对称，一旦记反就全错。

## 2.2 核心知识块 A：交通信号灯

### A1 信号灯基础知识

**交通信号灯由红灯、绿灯和黄灯组成**，这是科目一的送分题。

除基本三色灯外，考试还会涉及：
- **箭头信号灯**：红/黄/绿箭头，按箭头方向分别控制对应方向车道
- **车道信号灯**：红色叉形灯、红色箭头灯和绿色箭头灯
- **铁路道口信号灯**：闪烁红灯交替亮起，表示火车接近，禁止通行
- **人行横道信号灯**：绿灯亮起行人可通过，绿灯闪烁表示即将变灯

**信号灯优先级（考试关键）**：

> **交通警察指挥 > 信号灯**

在路口遇交通信号灯和交通警察指挥不一致时，**按交通警察指挥通行**，而不是按信号灯。这也是高频考点。

### A2 各色信号灯通行规则

<Tabs>
<TabItem value="red" label="红灯">

| 灯型 | 规则 |
|------|------|
| 圆形红灯 | 禁止通行。右转弯的车辆在不妨碍被放行车辆、行人通行的情况下，**可以通行**（无"禁止右转"辅助标志时） |
| 红色箭头灯 | 表示箭头所指方向**禁止通行** |
| 红色叉形灯 | 该车道**禁止**车辆进入 |
| 铁路道口红灯 | 两个红灯交替闪烁或一个红灯亮，**禁止**车辆和行人通行 |

> **为什么圆形红灯可以右转？**
>
> 右转不穿越对向车道，对通行秩序干扰最小，所以法律允许多数路口圆形红灯时右转。但注意，如果路口有"红灯禁止右转"辅助标志，或者右转箭头红灯亮，则不能右转。

</TabItem>
<TabItem value="green" label="绿灯">

| 灯型 | 规则 |
|------|------|
| 圆形绿灯 | 准许通行。转弯车辆**不得妨碍**直行车辆和行人通行 |
| 绿色箭头灯 | 准许车辆按箭头所指方向通行 |
| 绿色箭头灯（车道信号） | 准许本车道车辆按指示方向通行 |

> **绿灯右转也要让行**：
>
> 即使路口是绿灯，右转车辆也必须让行正在通行的行人和非机动车。这体现了"安全优先"的原则。

</TabItem>
<TabItem value="yellow" label="黄灯">

| 灯型 | 规则 |
|------|------|
| 黄灯持续亮 | 已越过停止线的车辆**可以继续通行**；未越过停止线的**应当停车** |
| 黄灯闪烁 | 提示车辆、行人注意瞭望，确认安全后通过 |

> **黄灯最容易错**：
>
> 很多学员以为"黄灯可以加速通过"，这是错的。黄灯亮时，车辆**未越过停止线就必须停**。只有在黄灯亮起前已经越过停止线的车辆才可以继续走。这一点在判断图中经常考——图中车尚未越过停止线但黄灯已亮，选项说"可以加速通过"就一定是错的。

</TabItem>
</Tabs>

### A3 信号灯常见陷阱

| 陷阱说法 | 正确结论 |
|----------|----------|
| "绿灯时可以随意加速通过" | 错，转弯车辆须让行直行和行人 |
| "黄灯闪烁时可以加速抢过" | 错，应减速瞭望，确认安全后通过 |
| "红色叉形灯亮表示此车道可以临时停车" | 错，表示禁止进入该车道 |
| "信号灯和交警指挥不一致时按信号灯走" | 错，应按交警指挥 |
| "箭头红灯亮时可以直行" | 错，箭头红灯对应方向禁止通行 |

## 2.3 核心知识块 B：交通标志

交通标志是本章**题量最大的板块**（约 370 题），也是学员翻车最密集的科目。根据 GB 5768.2，交通标志分为七大类别。

### B1 警告标志——"前方有危险，请注意"

| 特征 | 内容 |
|------|------|
| **颜色** | 黄底、黑边、黑图案 |
| **形状** | 正三角形（顶角朝上） |
| **含义** | 提醒车辆、行人注意前方危险或特殊情况 |

> **记忆口诀**：黄牌子都是提醒你的，不是禁止你的。

**高频考察的警告标志**：

| 标志典型特征 | 名称 | 考试易错点 |
|-------------|------|-----------|
| 三角形内有火车图案 | 无人看守铁路道口 | 区分"有人看守"（栅栏图案）和"无人看守"（火车图案） |
| 三角形内有人行走图案 | 注意行人 | 与人行横道标志（蓝底矩形）区分 |
| 三角形内有儿童图案 | 注意儿童 | 常出现在学校区域附近，注意观察减速 |
| 三角形内有感叹号 | 注意危险 | 其他警告标志无法表达的综合性危险 |
| 三角形内有急弯箭头 | 急转弯/连续弯 | 箭头方向和个数决定左/右/连续弯 |

> **警告标志 ≠ 禁令标志**：
>
> 警告标志不是让你不许走，而是提醒你前方什么情况，让你提前准备减速。题目常给一张黄三角图，选项里混入"禁止""必须"等指令，这类选项就是陷阱。

### B2 禁令标志——"禁止你做某事"

| 特征 | 内容 |
|------|------|
| **颜色** | 白底、红圈、红斜杠、黑图案 |
| **形状** | 圆形（少数为八角形或倒三角） |
| **含义** | 禁止或限制车辆、行人的交通行为 |

**禁令标志是标志题中扣分最密集的领域**，以下三组是最容易混淆的：

**第一组：停车禁令三兄弟**

| 标志特征 | 名称 | 含义 |
|----------|------|------|
| 红圈+红叉 | 禁止车辆停放 | 临时停放和长时间停放**都不允许** |
| 红圈+红斜杠 | 禁止车辆长时间停放 | 可以**临时停**，但不许长时间停放 |
| （对比）蓝底+波折线 | 允许临时停车 | 仅允许即停即走 |

> **判断技巧**：叉 = 全禁止（交叉 = 都不能停），斜杠 = 部分禁止（只禁长停）。考试最常把"禁止停放"和"禁止长时间停放"放在四个选项里让你挑，就看你能不能区分红叉和红斜杠。

**第二组：速度限制牌**

| 标志特征 | 含义 |
|----------|------|
| 红圈+数字 | 最高限速——不得超过该速度 |
| 蓝底+数字 | 最低限速——不得低于该速度 |
| 黑圈+黑斜杠 | 解除限制速度 |

> **记忆方法**：红色是"不能超过"（危险，限上限），蓝色是"不能低于"（安全，保下限）。

**第三组：让行标志二人转**

| 标志特征 | 名称 | 含义 |
|----------|------|------|
| 八角形+停字 | 停车让行 | 必须先停车观察，确认安全再通过 |
| 倒三角+让字 | 减速让行 | 必须减速，让干路车辆先行 |

> **"让"和"停"怎么分？**
>
> — 遇到"停"字：必须把车停下来，无论有没有来车。
> — 遇到"让"字：减速即可，如有来车则让，无来车可以慢速通过。
>
> 考试常拿"减速让行标志前应该停车观察"来迷惑——错，"让"只要减速就行。

### B3 指示标志——"你应该这样做"

| 特征 | 内容 |
|------|------|
| **颜色** | 蓝底、白图案 |
| **形状** | 圆形或矩形 |
| **含义** | 指示车辆、行人必须遵守的规定 |

**高频考察的指示标志**：

| 标志典型特征 | 名称 | 考试易错点 |
|-------------|------|-----------|
| 蓝底+直行箭头 | 直行 | 表示该车道只能直行，不能转弯或掉头 |
| 蓝底+左转箭头 | 向左转弯 | 该车道仅可左转 |
| 蓝底+直行左转双箭头 | 直行和向左转弯 | 该车道可直行和左转 |
| 蓝底+向右转弯箭头 | 向右转弯 | 该车道仅可右转 |
| 蓝底+自行车图案 | 非机动车行驶 | 区别于"注意非机动车"（黄三角警告标志） |
| 蓝底+行人图案 | 人行横道 | 区别于"注意行人"（黄三角警告标志） |
| 蓝底+直行小车图案 | 单行路（直行） | 只能向前行驶，不能倒车或掉头 |

### B4 指路标志——"路在哪个方向"

| 特征 | 内容 |
|------|------|
| **颜色** | 一般道路蓝底白字、高速公路绿底白字 |
| **形状** | 一般为矩形 |
| **含义** | 传递道路方向、地点、距离信息 |

> **考试涉及较少**，主要了解：蓝底是一般道路，绿底是高速公路；地名标牌上的数字代表距离（km）。

### B5 其他标志类别

| 类别 | 特征 | 考试频率 |
|------|------|----------|
| 旅游区标志 | 棕底白字/白图案 | 低 |
| 道路施工安全标志 | 多种颜色，临时设置 | 中（施工路段减速让行是常考点） |
| 辅助标志 | 附设在主标志下方，补充说明 | 中（如"200m""前方学校"等距离/范围/原因补充） |

### B6 标志易混淆对比速查

| 易混淆对 | 区分要点 |
|----------|----------|
| 注意行人（黄三角） vs 人行横道（蓝矩形） | 黄色 = 警告前方有行人出没；蓝色 = 指示此处是人行横道 |
| 注意儿童（黄三角） vs 学校区域（文字牌） | 黄色 = 警告注意儿童；蓝色文字 = 指示学校区域 |
| 禁止停车（红叉） vs 禁止长时间停车（红斜杠） | 叉 = 完全禁止；斜杠 = 不禁临时 |
| 最高限速（红圈） vs 最低限速（蓝底） | 红 = 上限；蓝 = 下限 |
| 停车让行（八角停字） vs 减速让行（倒三角让字） | 停 = 必须停车；让 = 减速即可 |

## 2.4 核心知识块 C：交通标线

根据 GB 5768.3，道路交通标线分为三类：**指示标线、警告标线、禁止标线**。考试约 78 题，重点集中在车道线、停止线和导向箭头。

### C1 标线三大分类

| 分类 | 作用 | 典型示例 |
|------|------|----------|
| **指示标线** | 指示车行道、行车方向、路面边缘等 | 车道分界线、导向箭头、人行横道线 |
| **禁止标线** | 告示道路交通的遵行、禁止、限制 | 禁止跨越对向车道线（黄实线）、停止线 |
| **警告标线** | 促使车辆驾驶人了解特殊情况 | 路面文字标记、减速标线 |

### C2 实线与虚线——标线中最核心的规则

| 线型 | 含义 | 考试结论 |
|------|------|----------|
| 白色虚线 | 同向车道分界线 | 可以变道 |
| 白色实线 | 同向车道分界线 / 导向车道线 | **不能变道** |
| 黄色虚线 | 双向车道分界线 | 在安全情况下可以借道超车 |
| 黄色实线（单/双） | 禁止跨越对向车道 | **绝对不能跨越** |
| 虚实线（黄） | 虚线侧可借道，实线侧不可 | 仅虚线一侧可临时跨越 |

> **核心原则**：实线 = 墙，虚线 = 帘。实线不能压、不能跨、不能越。虚线可以在安全前提下跨越。
>
> 考试最常考的变道规则：**交叉路口前变道必须在虚线区，按导向箭头指示变更**；进入实线区后不得变道。

### C3 导向箭头——"地上画的比你想象的更重要"

白色导向箭头画在路口前方车道上，指示该车道的行驶方向。高频考点：

| 箭头组合 | 含义 |
|----------|------|
| 单直行箭头 | 仅可直行 |
| 单左转箭头 | 仅可左转 |
| 单右转箭头 | 仅可右转 |
| 直行+左转箭头 | 可直行和左转 |
| 直行+右转箭头 | 可直行和右转 |
| 左右双箭头 | 仅可左右转弯 |

> **导向箭头 + 实线 = 铁律**：
>
> 一旦进入标有实线的导向车道，就必须按箭头方向行驶，不能变道、不能掉头。考试情景题里经常出现"车在直行导向车道实线区却打左转灯想转弯"的情况——答案是不得左转，只能直行。

### C4 停止线与人行横道预告

| 标线 | 特征 | 考试要点 |
|------|------|----------|
| 停止线 | 白色单实线，与道路垂直 | 红灯/黄灯亮时不得越过 |
| 人行横道预告标线 | 白色菱形图案，出现两次 | 看到菱形 = 前方有人行横道，应减速 |

> **白色菱形 = 前方斑马线**：
>
> 路面出现◇（菱形）图案，说明前方不远处就是人行横道，驾驶员应该提前减速、注意观察。考试判断题常把"看到菱形加速通过"设为错误选项。

### C5 标线常见陷阱

| 陷阱说法 | 正确结论 |
|----------|----------|
| "白色实线可以变道" | 错，实线不可变道 |
| "黄虚线绝对不能跨" | 错，虚线在安全情况下可以跨越 |
| "路口前任何位置都可以变道" | 错，只能在虚线区按导向箭头变道 |
| "进入导向车道实线后还能换方向" | 错，进入实线区必须按箭头方向行驶 |
| "黄色双实线可以掉头" | 错，双黄实线绝对不能跨越 |

## 2.5 核心知识块 D：交警手势

科目一通常考查 **8 种标准交警手势信号**，约 21 题。题量不大，但属于"错一个就全错"的高危题型——动作相似、左右对称，记混概率极高。

### D1 手势优先级与总体原则

**原则一：交警指挥优先级最高**

当路口既有信号灯又有交警指挥时，**以交警手势为准**。这是高频判断题，选项说"按信号灯通行"必错。

**原则二：先看方位，再看手臂方向**

交警手势要先判断交警与车辆的相对方位，再看手臂的**位置、角度和摆动方向**；不能只凭"脸朝哪边"或单只手臂孤立判断。

### D2 8 种手势速查表

| # | 手势名称 | 手臂动作特征 | 含义 |
|---|----------|-------------|------|
| 1 | **停止信号** | 左臂向前上方直伸，掌心向前 | 不准前方车辆通行 |
| 2 | **直行信号** | 左臂向左平伸、右臂向右平伸，右臂向左摆动 | 准许右方直行车辆通行 |
| 3 | **左转弯信号** | 右臂向前平伸，左臂向右前方摆动 | 准许车辆左转弯 |
| 4 | **左转弯待转信号** | 左臂向左下方45度平伸，掌心向下；左臂向下摆动 | 左转车辆进入待转区等待 |
| 5 | **右转弯信号** | 左臂向前平伸，右臂向左前方摆动 | 准许车辆右转弯 |
| 6 | **变道信号** | 右臂向前平伸，掌心向左；右臂向左水平摆动 | 示意车辆变更车道 |
| 7 | **减速慢行信号** | 右臂向右前方平伸，掌心向下；向下摆动至腰部 | 示意车辆减速慢行 |
| 8 | **靠边停车信号** | 左臂向前上方平伸，掌心向前；右臂向前下方平伸，向左水平摆动 | 示意车辆靠边停车 |

### D3 左右转弯手势——攻破最大难点

左转和右转手势的骨架高度对称，是考试中的"死亡题"：

| | 左转弯信号 | 右转弯信号 |
|------|----------|------------|
| **哪只手平伸** | **右臂**向前平伸 | **左臂**向前平伸 |
| **哪只手摆动** | **左臂**向右前方摆动 | **右臂**向左前方摆动 |
| **脸朝** | 正前方 | 正前方 |

> **记忆口诀：左转——"右定左摆"，右转——"左定右摆"**
>
> 哪个方向转弯，就用**相反的手臂平伸**作为"定杆"，另一只手臂摆动给出转弯信号。不要把摆动方向简单理解成"往左就是左转、往右就是右转"，考试图要结合交警方位判断。
>
> 一个形象的联想：**左转 = 右手扶门框，左手招呼向左拐；右转 = 左手扶门框，右手招呼向右拐**。

### D4 手势常见陷阱

| 陷阱 | 正解 |
|------|------|
| "停止信号"和"靠边停车信号"都有左臂向前上方伸——混淆 | 看右手：只有左臂 = 停止；右臂前下方平伸并向左摆动 = 靠边停车 |
| "左转弯待转"和"减速慢行"手臂下垂方向相似——混淆 | 左转弯待转：左臂向下摆动，减速慢行：右臂向下摆动 |
| "直行信号"和"变道信号"都有手臂前伸——混淆 | 直行：双臂平伸；变道：仅右臂前伸摆动 |
| 左右转弯记反 | 左转 = 右臂平伸 + 左臂摆动；右转 = 左臂平伸 + 右臂摆动，具体以交警方位和图示为准 |

## 本章小结

交通信号是驾驶员与道路"对话"的语言。信号灯告诉你什么时候走、什么时候停；标志告诉你这条路能做什么、不能做什么；标线告诉你车道怎么走、不能怎么走；交警手势则是整个体系中的最高指令——当它们发生冲突，交警说啥就是啥。

科目一考试中，本章有大量的图文结合题，考查的不是死记硬背，而是你的**图形识别能力和规则应用能力**。学会在标志标线中寻找颜色和形状的规律（黄色 = 警告、红色 = 禁令、蓝色 = 指示），而非死记每个标志的描述，是通关这章的捷径。

**最高频的丢分点**：禁令标志中的"停车三兄弟"（禁止停放 vs 禁止长时间停放）、让行标志的"停 vs 让"、左右转弯手势的对称记忆，以及标线中"实线不能跨"的原则。把这四个硬骨头啃下来，本章就过了大半。
