# 第零章 考试概况与学习路线

## 0.1 考试基本信息

| 项目 | 说明 |
|------|------|
| 考试名称 | 科目一 道路交通安全法律、法规和相关知识考试 |
| 适用车型 | C1（小型汽车） |
| 考试方式 | 计算机无纸化考试 |
| 题量 | 100 题 |
| 题型 | 判断题 + 单选题 |
| 考试时间 | 45 分钟 |
| 合格标准 | 90 分（含）以上 |
| 每题分值 | 1 分 |

> **注**：C1 科目一只包含判断题和单选题。多选题是科目三（安全文明驾驶常识）的题型，不在科目一考核范围内。

## 0.2 题型构成

```
判断题（约 40 题）  ← 每题 1 分，"对"或"错"二选一
单选题（约 60 题）  ← 每题 1 分，四个选项中选一个
────────────────────
总计 100 题，满分 100 分
```

### 判断题策略

判断题的命题通常有两类：

- **绝对化表述**：题干出现"一律""必须""只要……就……"等词语时，多数为错——但并非绝对，某些规定确实是强制性的
- **条件缺漏**：题干只说了半句，省去了关键前提条件，容易误判为对

> **不要依赖"题感"蒙**：判断题的"绝对化词一定有误"这类技巧只能应付少量题。重点还是建立正确的知识判断标准。

### 单选题策略

单选题的关键是**排除法**：

1. 先排除明显错误的选项（通常有 1-2 个）
2. 在剩下的选项中，找到与法条规定完全一致的表述
3. 如果两个选项都看起来"有点像"，回到法条原文对比措辞

## 0.3 考试范围（依据公安部令第 172 号）

科目一考试内容包括：

1. **道路交通安全法律、法规和规章**
   - 驾驶证申领与使用规定
   - 交通违法行为记分制度
   - 交通违法处罚
   - 交通事故处理程序
   - 机动车登记规定

2. **交通信号**
   - 道路交通信号灯
   - 交通标志
   - 交通标线
   - 交通警察手势

3. **安全行车、文明驾驶基础知识**
   - 一般道路行车规则
   - 高速公路行车规则
   - 恶劣天气和复杂道路条件
   - 紧急情况处置
   - 文明驾驶与礼让

4. **机动车驾驶操作相关基础知识**
   - 仪表与指示灯
   - 操纵装置
   - 安全装置
   - 日常检查维护

## 0.4 题库概览

本手册使用的典型例题来自第三方题库 [doupoa/DrivingTestSubjectOne](https://github.com/doupoa/DrivingTestSubjectOne)，共收录题目约 **2545 题**。

各知识域大致占比：

| 知识域 | 约占比例 | 难度 |
|--------|----------|------|
| 法律·法规·规章 | ~40% | ★★★★ |
| 交通信号 | ~20% | ★★★ |
| 安全行车·文明驾驶 | ~25% | ★★★ |
| 驾驶操作基础 | ~12% | ★★ |
| 其他/地方性规定 | ~3% | ★ |

> **最重要的发现**：法律·法规·规章是题量最大、丢分率最高的章节，也是你投入时间回报率最高的地方。

## 0.5 三类失分模式

了解自己最容易在哪里丢分，才能精准发力：

### 模式一：数字混淆

科目一的"数字题"是最大失分来源——记分、罚款、速度、年限、年龄……各种数字混杂在一起。

**对策**：不要零散记忆，用「横向打通」的方式——把所有 12 分的行为放在一起看，所有 2000 元以上罚款放在一起看。

### 模式二：概念混淆

暂扣、吊销、撤销、注销——四个看起来差不多的词，法律后果完全不同。

**对策**：本手册每章配有**对比辨析表**，把易混淆概念放在一起对比。

### 模式三：场景误判

题目给一个驾驶场景，问"该怎么做"。很多人凭日常经验判断，但法律规定可能和经验相反。

**对策**：本手册对每种高频场景提供**逻辑推理链**，告诉你判题的"步骤"，而非结果。

## 0.6 三步学习法

```
第一步：通读知识核心（第1-4章）
  → 建立知识框架，理解"为什么这样规定"
  → 时间：总学习时间的 40%

第二步：专项突破（横向打通）
  → 把扣分、罚款、速度等数字题集中攻克
  → 时间：总学习时间的 30%

第三步：仿真刷题 + 查漏补缺
  → 在驾考宝典/驾校一点通做仿真，错题回查手册
  → 时间：总学习时间的 30%
```

> **手册 vs 刷题 App 的分工**：本手册帮你**理解**，刷题 App 帮你**熟练**。不要只看手册不刷题，也不要只刷题不翻手册。

## 本章小结

1. C1 科目一考 100 道判断题和单选题，90 分合格，45 分钟完成
2. 法律·法规·规章是最大分值区，也是最大丢分区
3. 数字混淆、概念混淆、场景误判是三种最主要的失分模式
4. 理解 + 对比 + 刷题 = 最高效的备考组合
