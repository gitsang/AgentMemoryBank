# C1 驾照科目一题库来源整理（2026-05-12）

## 结论先说

目前没有检索到一个公开、可下载、由公安部或 12123 官方发布的「C1 科目一完整最新考题库」。官方公开的是考试范围、法规依据、考试预约/成绩等服务，不是完整题库导出。

备考时建议把资料分成两类用：

1. **官方法规与平台**：用来确认规则是不是最新、答案有没有过期。
2. **商业/第三方题库**：用来刷题、做模拟、记错题。它们不能等同于官方题库，但覆盖度通常更适合备考。

## 不建议直接信的来源

- 标题写「官方 500 题」「2026 完整 1073 题」「车管所内部题库」的文库/PDF/下载站，很多没有来源链路，甚至引用的是旧版公安部令第 91、111、162 号。
- 要求下载不明 APK、扫码答题神器、所谓「12123 驾考题库」但开发者不是公安部交通管理科学研究所的 App。
- 只给 10 道快测、正文大量堆关键词的页面。它们可作练习，不适合作为判断“最新法规答案”的依据。

## 建议的整理方式

按官方考试范围，把题目归到 5 个篮子里：

1. **道路交通安全法律、法规和规章**：驾驶证申领使用、违法记分、处罚、事故处理。
2. **交通信号**：信号灯、标志、标线、交警手势。
3. **安全行车、文明驾驶基础知识**：让行、会车、超车、恶劣天气、高速公路。
4. **机动车驾驶操作基础**：仪表盘、灯光、ABS、安全带、车辆故障。
5. **地方性规定/新规变化**：以所在省市车管所、12123 提示、当地交管局公告为准。

C1/C2 备考不必刷货车、客车、摩托车专用题。第三方题库里如果混入 A/B/D/E/F 专用题，直接跳过。

## 实操路线

1. 在「驾考宝典」或「驾校一点通」选 **小车 C1/C2/C3、科目一**。
2. 顺序练习刷完一遍，错题全部收藏。
3. 按章节补弱项：扣分题、交通标志、仪表灯、高速/恶劣天气通常最容易错。
4. 考前 3 天每天做 3-5 套 100 题仿真，稳定 95 分以上再约考更稳。
5. 遇到扣分、年龄、换证、登记、满分学习这类题，如果 App 之间答案不一致，回到上面的官方法规链接核对。

## 当前判断

最稳组合是：**官方法规核对 + 驾考宝典/驾校一点通主刷 + 元贝或驾照网补充**。不要花钱买来历不明的「完整官方题库」，它通常既不官方，也未必比主流 App 更新。
