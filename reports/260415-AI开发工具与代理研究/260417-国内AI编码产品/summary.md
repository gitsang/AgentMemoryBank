# Mainland China AI coding products shortlist

Date: 2026-04-17

## Bottom line

For mainland China users, the most stable paid AI coding options are the products with native China billing or Chinese API providers:

1. Tongyi Lingma
2. Tencent CodeBuddy
3. DeepSeek API + OpenCode/Cline/Continue
4. Z.AI / GLM API + OpenCode
5. Moonshot Kimi API + OpenCode
6. MiniMax API + OpenCode
7. OpenCode Go / Zen

Products like Claude Code, direct ChatGPT Plus/Pro, and direct Anthropic subscriptions are not stable options for mainland China users because of official region restrictions.

## Recommended shortlist by scenario

### A. Want the most stable official domestic payment

- Tongyi Lingma
- Tencent CodeBuddy
- Baidu Comate

Why: domestic vendors, RMB billing, mainstream local payment methods.

### B. Want the strongest flexibility for coding workflows

- OpenCode + DeepSeek
- OpenCode + Z.AI
- OpenCode + Moonshot Kimi
- OpenCode + MiniMax

Why: lower lock-in, cheaper than many bundled subscriptions, easier to swap models.

### C. Want a low-cost packaged subscription

- OpenCode Go
- Trae Pro

Why: lower monthly cost and easier onboarding than assembling a western subscription stack.

## Product notes

### Tongyi Lingma

- Good for: teams wanting stable official China support
- Payment: native domestic payment support
- Tradeoff: less flexible than BYOK toolchains

### Tencent CodeBuddy

- Good for: users wanting an IDE assistant with RMB pricing
- Payment: native domestic payment support
- Tradeoff: more product lock-in than open BYOK tools

### DeepSeek

- Good for: best cost/performance API option
- OpenCode: native provider / easy integration
- Tradeoff: API workflow, not a polished bundled IDE by itself

### Z.AI / GLM

- Good for: domestic API option with OpenCode support
- OpenCode: native provider
- Tradeoff: model ecosystem differs from western defaults

### Moonshot Kimi

- Good for: strong Chinese provider option via OpenAI-compatible API
- OpenCode: custom provider using OpenAI-compatible base URL

### MiniMax

- Good for: another domestic OpenAI-compatible provider
- OpenCode: custom provider using OpenAI-compatible base URL

### OpenCode Go / Zen

- Good for: users who want OpenCode-first workflow without assembling providers manually
- OpenCode: native
- Tradeoff: still depends on OpenCode-hosted gateway rather than direct domestic provider ownership

## Products to treat as high-risk / low-stability

- Claude Code direct subscription
- ChatGPT Plus / Pro direct subscription
- Direct Anthropic web subscriptions
- Cursor if your payment path depends on unofficial resellers or unsupported cards

Reason: official region/payment restrictions or unstable payment success for mainland users.

## OpenCode compatibility summary

- Native in OpenCode: DeepSeek, Z.AI, OpenCode Zen, OpenCode Go, OpenRouter, GitHub Copilot, Bedrock, Vertex AI, 302.AI, local Ollama/LM Studio
- OpenAI-compatible custom config: Moonshot, MiniMax
- Best mainland-friendly OpenCode stack:
  - DeepSeek
  - Z.AI
  - Moonshot
  - MiniMax
  - OpenCode Go / Zen

## Practical recommendation

If you want the safest answer for mainland China users:

1. Domestic packaged product: Tencent CodeBuddy or Tongyi Lingma
2. Most flexible serious workflow: OpenCode + DeepSeek or Z.AI
3. Cheapest viable entry: OpenCode Go or a domestic API + OpenCode
