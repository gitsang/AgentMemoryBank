# C1 科目一备考指南

欢迎使用 C1 科目一备考系统。本指南帮助你从零开始，系统化地准备 C1 驾驶证科目一理论考试。

## 核心原则

1. **刷题是王道。** 2,545 题全部刷完是底线，错题反复刷是核心。
2. **模考定生死。** 连续 5 套 90+ 分才能预约考试，90-91 分不算稳。
3. **数字要死记。** 速度、距离、罚款、扣分——这些没有捷径。
4. **图片多对比。** 相似标志放一起对比记忆，比单独记有效得多。
